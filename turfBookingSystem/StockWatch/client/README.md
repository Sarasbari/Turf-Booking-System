### 📘 Turf Booking System

A full-stack web application built for managing turf bookings efficiently. This system allows users to view turf availability, book slots, manage their reservations, and receive confirmation — designed with simplicity and usability in mind for college sports and recreational purposes.

---

### 🔧 Features:

* 🕒 Real-time slot availability
* 📝 User registration & login
* ✅ Booking confirmation with time and date
* 📆 Admin panel for managing turfs and bookings
* 🔒 Authentication and validation
* 📱 Fully responsive design

---

### 🛠️ Tech Stack:

* **Frontend**: HTML, CSS, JavaScript 
* **Backend**: Node.js, Express.js
* **Database**: postgressSQL

---

### 🎯 Goal:

To streamline the process of booking college turfs for students and faculty by reducing manual intervention and ensuring fair access through a user-friendly digital platform.
